La grammaire est l'ensemble des règles qui régissent la structure d'une langue. Elle couvre plusieurs aspects comme les règles de **conjugaison**, **accords**, **types de phrases**, **temps verbaux**, etc. Comprendre et appliquer correctement ces règles est essentiel pour formuler des phrases claires et correctes.

# Les Parties du Discours

## Le Nom (ou Substantif)

Un nom désigne une personne, un lieu, un objet ou une idée.

-   **Exemple** : *chat*, *liberté*, *Paris*, *voiture*.
-   **Exemple dans une phrase** : "Le **chat** dort sur la chaise."

## L'Adjectif

L'adjectif qualifie le nom en précisant ses caractéristiques (forme, couleur, taille, etc.).

-   **Exemple** : *grand*, *rouge*, *intelligent*.
-   **Exemple dans une phrase** : "Le chat **noir** dort sur la chaise."

## Le Verbe

Le verbe indique une action ou un état.

-   **Exemple** : *manger*, *être*, *parler*.
-   **Exemple dans une phrase** : "Le chat **dort** sur la chaise."

## Le Pronom

Le pronom remplace un nom pour éviter les répétitions.

-   **Exemple** : *il*, *nous*, *le*, *cela*.
-   **Exemple dans une phrase** : "**Il** dort sur la chaise."

## L'Adverbe

L'adverbe modifie un verbe, un adjectif ou un autre adverbe.

-   **Exemple** : *vite*, *bien*, *toujours*.
-   **Exemple dans une phrase** : "Le chat dort **calmement**."

## La Préposition

La préposition établit une relation entre deux éléments dans une phrase.

-   **Exemple** : *à*, *dans*, *sur*, *par*.
-   **Exemple dans une phrase** : "Le chat dort **sur** la chaise."

## La Conjonction

La conjonction relie des phrases ou des mots.

-   **Exemple** : *et*, *mais*, *ou*, *car*.
-   **Exemple dans une phrase** : "Le chat dort, **mais** il se réveillera bientôt."

## L'Interjection

L'interjection exprime une émotion ou une réaction spontanée.

-   **Exemple** : *Oh!*, *Aïe!*, *Bravo!*.
-   **Exemple dans une phrase** : "**Oh!** Le chat s'est réveillé!"

# Les Accords

## L'Accord du Verbe avec le Sujet

Le verbe s'accorde en nombre (singulier/pluriel) et en personne avec son sujet.

● **Exemple** :

○ *Le chat dort* (le verbe "dort" s'accorde avec "le chat", sujet singulier).

○ *Les chats dorment* (le verbe "dorment" s'accorde avec "les chats", sujet pluriel).

## L'Accord de l'Adjectif avec le Nom

L'adjectif s'accorde en genre (masculin/féminin) et en nombre avec le nom qu'il qualifie.

● **Exemple** :

○ *Un chat noir* (adjectif masculin singulier).

○ *Une voiture rouge* (adjectif féminin singulier).

○ *Des chats noirs* (adjectif masculin pluriel).

○ *Des voitures rouges* (adjectif féminin pluriel).

## L'Accord du Participe Passé

L'accord du participe passé dépend de l'auxiliaire utilisé et de la présence ou non d'un complément d'objet direct (COD).

-   Avec **l'auxiliaire "avoir"** : Le participe passé s'accorde avec le COD **si** celui-ci est placé avant le verbe.
    1.  **Exemple** :

■ *J'ai mangé une pomme.* (pas d'accord, COD après le verbe).

■ *La pomme que j'ai mangée était délicieuse.* (accord avec "pomme", COD placé avant le verbe).

-   Avec **l'auxiliaire "être"** : Le participe passé s'accorde toujours avec le sujet.
    1.  **Exemple** :

■ *Elle est allée au marché.*

■ *Ils sont partis hier.*

# Les Types de Phrases

## Phrase Déclarative

Exprime une information ou un fait.

● **Exemple** : "Le chat dort sur la chaise."

## Phrase Interrogative

Pose une question.

● **Exemple** : "Où est le chat ?"

## Phrase Exclamative

Exprime une émotion ou une surprise.

● **Exemple** : "Quel beau chat !"

## Phrase Impérative

Donne un ordre ou une instruction.

● **Exemple** : "Donne à manger au chat."

# Les Temps Verbaux

## Présent de l'Indicatif

Exprime une action en cours ou un état présent.

● **Exemple** : "Je mange une pomme."

## Passé Composé

Exprime une action achevée dans le passé.

● **Exemple** : "J'ai mangé une pomme hier."

## Futur Simple

Exprime une action future.

● **Exemple** : "Je mangerai une pomme demain."

## Imparfait

Exprime une action passée qui dure ou une habitude dans le passé.

● **Exemple** : "Quand j'étais petit, je mangeais une pomme chaque jour."

## Plus-que-parfait

Exprime une action antérieure à une autre action passée.

● **Exemple** : "Quand je suis arrivé, elle avait déjà mangé."

# La Négation

La négation s'exprime généralement avec "ne" suivi d'un autre élément négatif comme "pas", "jamais", "rien", etc.

-   **Exemple** :

○ "Je **ne** veux **pas** sortir."

○ "Il **ne** mange **jamais** de légumes."

**Exception** : Le "ne" disparaît parfois dans le langage familier.

-   **Exemple** : "Je veux pas sortir."

# Les Pronoms Relatifs

Les pronoms relatifs servent à relier deux propositions. Ils incluent : **qui**, **que**, **dont**, **où**.

● **Exemple** :

○ "Le livre **que** je lis est passionnant."

○ "La ville **où** je suis né est magnifique."

# Les Modes Verbaux

## L'Indicatif

Exprime des actions réelles ou certaines.

● **Exemple** : "Il mange une pomme."

## Le Subjonctif

Exprime une action incertaine, souhaitée ou envisagée.

● **Exemple** : "Il faut que tu **manges**."

## Le Conditionnel

Exprime une action qui dépend d'une condition.

● **Exemple** : "Si j'avais faim, je **mangerais** une pomme."

## L'Impératif

Exprime un ordre ou une demande.

● **Exemple** : "Mange une pomme !"

# Les Accord de Liaisons

Les liaisons sont des phénomènes où la dernière consonne d'un mot est prononcée quand il est suivi d'un mot commençant par une voyelle.

● **Exemple** :

○ *Les amis* → liaison prononcée : "z" entre "les" et "amis".

○ *On est* → pas de liaison.

# Les Particularités

## Les Homophones

Les homophones sont des mots qui se prononcent de la même manière mais ont des significations différentes. Il est important de bien les distinguer pour éviter les erreurs.

● **Exemples** :

○ *Son* (possessif) et *sont* (verbe être).

○ *Ce* (déterminant) et *se* (pronom réfléchi).

## Les Elisions

L'élision est la suppression de la voyelle finale de certains mots devant un mot commençant par une voyelle ou un "h" muet.

● **Exemple** :

○ *Je aime* devient *j'aime*.

○ *Le arbre* devient *l'arbre*.
