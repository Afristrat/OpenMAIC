| **Paramètre**               | **Exemples**                                        | **Phrase Exemple**                       |
|-----------------------------|-----------------------------------------------------|------------------------------------------|
| **1. Connecteurs logiques** | Addition : et, Opposition : mais, Cause : parce que | "Il pleut **et** il fait froid."         |
| **2. Adverbes**             | Temps : demain, Lieu : ici, Manière : lentement     | "Je partirai **demain**."                |
| **3. Conjonctions**         | Coordination : mais, Subordination : lorsque        | "Je viendrai **lorsque** tu seras prêt." |
| **4. Prépositions**         | Lieu : dans, Temps : pendant, Cause : à cause de    | "Je suis **dans** la maison."            |
| **5. Pronoms relatifs**     | Qui, que, dont                                      | "L’homme **qui** parle est mon ami."     |
| **6. Pronoms personnels**   | Je, tu, il, elle                                    | "**Je** suis fatigué."                   |
| **7. Déterminants**         | Défini : le, la, les, Indéfini : un, une, des       | "**Le** chat dort."                      |
| **8. Verbes pronominaux**   | Se laver, se lever                                  | "Il **se lave** tous les matins."        |
| **9. Verbes modaux**        | Pouvoir, devoir, savoir                             | "Il **peut** venir demain."              |
| **10. Verbes transitifs**   | Aimer, manger                                       | "Il **mange** une pomme."                |
| **11. Verbes intransitifs** | Partir, venir                                       | "Elle **part** demain."                  |
| **12. Accord sujet-verbe**  | Je suis, nous sommes                                | "**Nous sommes** prêts."                 |
| **13. Accord adjectif**     | Féminin : belle, Masculin : beau                    | "Elle est **belle**."                    |
| **14. Articles définis**    | Le, la, les                                         | "**La** maison est grande."              |
| **15. Articles indéfinis**  | Un, une, des                                        | "**Un** chat dort sur le canapé."        |
| **16. Noms communs**        | Maison, chat, livre                                 | "**Le chat** dort."                      |
| **17. Noms propres**        | Marie, Paris, Louvre                                | "**Marie** est allée à Paris."           |

| **18. Phrases interrogatives**    | Est-ce que, pourquoi, comment  | "**Pourquoi** es-tu parti ?"                |
|-----------------------------------|--------------------------------|---------------------------------------------|
| **19. Phrases impératives**       | Viens, écoute, regarde         | "**Écoute** bien ce que je dis."            |
| **20. Accord du participe passé** | Mangé, vu, pris                | "Elle a **mangé** une pomme."               |
| **21. Conditionnel**              | Aurais, pourrais, serais       | "Si j’avais su, je **serais** venu."        |
| **22. Futur simple**              | Irai, ferai, serai             | "Je **ferai** le travail demain."           |
| **23. Passé composé**             | Ai fait, suis venu             | "Il **a fait** ses devoirs."                |
| **24. Subjonctif présent**        | Que je sois, que tu viennes    | "Il faut que **tu viennes** avec nous."     |
| **25. Gérondif**                  | En mangeant, en parlant        | "Il a répondu **en mangeant**."             |
| **26. Infinitif présent**         | Manger, parler                 | "**Manger** est essentiel."                 |
| **27. Négations**                 | Ne…pas, ne…jamais              | "Il **ne** veut **pas** partir."            |
| **28. Phrases exclamatives**      | Quel, comme, que               | "**Comme** tu as grandi !"                  |
| **29. Comparaison**               | Plus que, aussi que, moins que | "Il est **plus grand que** toi."            |
| **30. Superlatifs**               | Le plus, le moins              | "C’est le **plus beau** film."              |
| **31. Adjectifs qualificatifs**   | Grand, petit, rapide           | "Un **grand** homme marche."                |
| **32. Adjectifs démonstratifs**   | Ce, cet, cette, ces            | "**Ces** fleurs sont belles."               |
| **33. Adjectifs possessifs**      | Mon, ton, son, notre, leur     | "**Mon** chat est noir."                    |
| **34. Forme passive**             | Est mangé, a été vu            | "Le gâteau **a été mangé**."                |
| **35. Pronoms** **interrogatifs** | Qui, quoi, lequel              | "**Qui** est venu hier ?"                   |
| **36. Phrases subordonnées**      | Parce que, comme, si           | "Il est venu **parce que** je l’ai invité." |

| **37. Prépositions de temps**         | Depuis, pendant, avant                | "Il dort **pendant** deux heures."           |
|---------------------------------------|---------------------------------------|----------------------------------------------|
| **38. Prépositions de lieu**          | Sur, sous, derrière, entre            | "Le chat est **sous** la table."             |
| **39. Possessifs**                    | Mon, ton, son, leur                   | "C’est **leur** maison."                     |
| **40. Conjonctions de cause**         | Parce que, puisque, car               | "Il pleure **parce que** il a mal."          |
| **41. Conjonctions de conséquence**   | Donc, ainsi, c’est pourquoi           | "Il est fatigué, **donc** il dort."          |
| **42. Hypothèse**                     | Si, au cas où                         | "**Si** tu viens, je te verrai."             |
| **43. Adverbes d’intensité**          | Très, beaucoup, trop                  | "Elle est **très** contente."                |
| **44. Temps composés**                | Avoir mangé, être allé                | "Elle **a été** chercher des fruits."        |
| **45. Forme emphatique**              | C’est… qui, c’est… que                | "**C’est toi qui** as fait cela !"           |
| **46. Apposition**                    | Jean, **le frère de Marie**, est venu | "**Jean, le frère de Marie**, est là."       |
| **47. Comparatifs**                   | Meilleur que, pire que                | "Il est **meilleur que** son frère."         |
| **48. Forme interrogative indirecte** | Je me demande si, pourquoi            | "Je me demande **pourquoi** il est parti."   |
| **49. Locutions conjonctives**        | Bien que, malgré que                  | "**Bien que** fatigué, il est parti."        |
| **50. Formule de politesse**          | S'il vous plaît, merci, excusez-moi   | "**S'il vous plaît**, pouvez-vous m'aider ?" |
